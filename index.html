Cài đặt kết nối dữ liệu
💡 Điền thông tin bên dưới để kết nối Sheets & AI thật. Hoặc dùng Demo để chạy thử ngay.
1. Google Sheet ID
Lấy từ URL Sheet: spreadsheets/d/ID/edit
2. Apps Script URL (để nhập liệu & gửi email)
Deploy Apps Script → Web app → Tất cả mọi người → copy URL exec
3. Anthropic API Key (để dùng AI)
Lấy tại console.anthropic.com — nếu đã có Apps Script URL thì bỏ qua (proxy qua Apps Script).
Dùng dữ liệu demo (không cần cài đặt)
📋 Hướng dẫn Apps Script:
1. Mở Google Sheet → Extensions → Apps Script
2. Dán code từ file ops-script.gs
3. Deploy → New deployment → Web app → Execute as: Me → Anyone
4. Copy URL → Dán vào ô số 2 bên trên
