Cài đặt kết nối
⚠️ Chỉ tài khoản System (Thanh Liêm) mới có thể thay đổi cài đặt.
1. Google Sheet ID
Lấy từ URL: spreadsheets/d/ID/edit
2. Apps Script URL
Deploy → Web app → Anyone → copy URL exec
3. Anthropic API Key
Lấy tại console.anthropic.com
Hướng dẫn Apps Script:
1. Sheet → Extensions → Apps Script → dán code ops-script-final.gs
2. Chạy initSheets() 1 lần → Deploy Web App → Anyone
3. Copy URL → Dán vào ô 2 bên trên
